Talent Pool Search

Talent Pool Search is a web application that helps recruiters search resumes.

Features

* Upload PDF and DOCX resumes
* Extract resume text
* Remove personal information before AI processing
* Extract skills, experience, job title and location using AI
* Store candidates in database
* Search by skill, location and experience

Tech Stack

* Python
* Flask
* SQLite
* Google Gemini API

Run Project

1. Create virtual environment
2. Activate virtual environment
3. Install requirements
4. Run:

python app.py

5. Open:

http://127.0.0.1:5000
